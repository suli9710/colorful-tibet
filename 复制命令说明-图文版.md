### 步骤3：精确复制命令

**方法A：手动选中（推荐）**

1. 找到这一行：`sudo bash -c 'cat > /tmp/server-setup.sh << '\''SCRIPT_EOF'\''`
   - 这是第17行
   
2. 用鼠标从这一行开始选中

3. 一直向下滚动，选中到这一行：`chmod +x /tmp/server-setup.sh && bash /tmp/server-setup.sh'`
   - 这是第160行

4. 按 `Ctrl + C` 复制

**方法B：使用查找功能**

1. 按 `Ctrl + F` 打开查找框
2. 搜索：`sudo bash -c`
3. 从找到的这一行开始，一直选中到文件末尾的 `bash /tmp/server-setup.sh'`
4. 按 `Ctrl + C` 复制

### 步骤4：登录腾讯云服务器

1. 打开浏览器
2. 访问：https://console.cloud.tencent.com/
3. 登录账号
4. 点击：**云服务器 CVM** → **实例**
5. 找到您的服务器，点击 **登录** 按钮
6. 选择 **标准登录方式**
7. 输入：
   - 用户名：`ubuntu`
   - 密码：您创建服务器时设置的密码
8. 点击 **登录**

### 步骤5：粘贴命令到终端

在腾讯云Web终端中：

1. **右键点击**终端窗口
2. 选择 **粘贴**（或按 `Ctrl + Shift + V`）
3. 您会看到命令被粘贴进去
4. 按 **回车键** 执行

## 🚨 常见问题

### Q1: 复制了说明文字怎么办？
**A:** 删除说明文字，只保留命令部分。或者重新复制，只复制命令。

### Q2: 在终端中粘贴后没有反应？
**A:** 
- 尝试右键点击终端 → 粘贴
- 或按 `Ctrl + Shift + V`（不是 `Ctrl + V`）

### Q3: 命令执行到一半就停止了？
**A:** 可能是复制不完整，重新复制完整的命令。

### Q4: 如何确认复制的内容正确？
**A:** 检查：
- 第一行是：`sudo bash -c 'cat > /tmp/server-setup.sh`
- 最后一行是：`chmod +x /tmp/server-setup.sh && bash /tmp/server-setup.sh'`
- 中间有很多行代码

## 💡 最简单的方法

如果您觉得上面的步骤太复杂，可以：

1. **打开文件**：`腾讯云一键安装命令.txt`

2. **全选复制**：按 `Ctrl + A`，然后 `Ctrl + C`

3. **粘贴到腾讯云终端**：右键 → 粘贴

4. **删除多余内容**：
   - 删除开头的说明文字（从文件开头到第一个 `=====` 之后）
   - 删除结尾的说明文字（从最后一个 `=====` 之后）
   - 只保留中间的命令部分

5. **执行**：按回车
