# 图片资源目录说明

## 目录结构

```
images/
├── spots/          # 景点图片
├── heritage/       # 非遗项目图片
├── news/           # 新闻图片
└── common/         # 通用图片（背景、占位符等）
```

## 图片访问路径

所有图片通过以下路径访问：
- 后端：`http://localhost:8080/images/...`
- 前端：`/images/...`（相对路径，会自动代理到后端）

## 图片命名规范

1. **景点图片**：使用景点名称，如 `布达拉宫.jpg`
2. **通用图片**：使用描述性名称，如 `potala-palace-hero.jpg`
3. **占位图片**：使用 `default-` 前缀，如 `default-news.jpg`

## 添加新图片

1. 将图片文件放到对应的目录
2. 在代码中使用 `/images/目录名/文件名` 路径
3. 确保图片文件名不包含特殊字符（建议使用中文或英文+连字符）

## 注意事项

- 图片格式支持：JPG、PNG、JPEG
- 建议图片大小：不超过 2MB
- 建议图片尺寸：1920x1080 或更小（根据用途调整）
